# Linear Design System

## 1. Visual Theme & Atmosphere

Linear is a project management tool for modern software teams. The design reflects speed, precision, and clarity. Every interaction feels instant and purposeful.

### Atmosphere
- Clean, minimal, purposeful
- No decorative elements — everything serves a function
- Dark-first with light mode support
- Monospace accents for technical feel

## 2. Color Palette

### Primary Colors
- **Purple**: #5E6AD2 (primary brand)
- **Purple Light**: #7B61FF
- **Purple Dark**: #4C3FB5

### Neutral Colors
- **Background**: #1B1B1B (dark) / #FFFFFF (light)
- **Surface**: #232323 (dark) / #F5F5F5 (light)
- **Border**: #333333 (dark) / #E5E5E5 (light)
- **Text Primary**: #E8E8E8 (dark) / #1A1A1A (light)
- **Text Secondary**: #8A8A8A (dark) / #666666 (light)

### Status Colors
- **Success**: #2DA44E
- **Warning**: #D4A72C
- **Error**: #CF222E
- **Info**: #5E6AD2

## 3. Typography

### Font Family
- **Primary**: Inter, -apple-system, BlinkMacSystemFont, sans-serif
- **Mono**: JetBrains Mono, SF Mono, monospace

### Scale
- **H1**: 28px / 34px / Bold
- **H2**: 22px / 28px / SemiBold
- **H3**: 16px / 24px / SemiBold
- **Body**: 14px / 20px / Regular
- **Small**: 12px / 16px / Regular
- **Mono**: 13px / 18px / Regular

## 4. Spacing & Layout

### Spacing Scale (4px base)
- **xs**: 4px
- **sm**: 8px
- **md**: 12px
- **lg**: 16px
- **xl**: 24px
- **2xl**: 32px
- **3xl**: 48px

### Border Radius
- **Small**: 4px
- **Medium**: 6px
- **Large**: 8px
- **Full**: 9999px

## 5. Components

### Buttons
- **Primary**: Purple background, white text, 8px radius
- **Secondary**: Transparent, purple border, purple text
- **Ghost**: Transparent, no border, text color

### Inputs
- Dark surface background
- 1px border
- 6px radius
- 12px padding

### Cards
- Surface background
- 1px border
- 8px radius
- 16px padding

## 6. Motion & Animation

### Easing
- **Default**: cubic-bezier(0.25, 0.1, 0.25, 1)
- **Entrance**: cubic-bezier(0, 0, 0.2, 1)
- **Exit**: cubic-bezier(0.4, 0, 1, 1)

### Duration
- **Instant**: 100ms
- **Fast**: 150ms
- **Normal**: 200ms
- **Slow**: 300ms

## 7. Voice & Tone

- Technical but approachable
- Concise, no filler
- Action-oriented
- Confident but not arrogant

## 8. Anti-patterns

- Don't use gradients
- Don't use drop shadows on cards
- Don't use decorative icons
- Don't animate for the sake of animation
