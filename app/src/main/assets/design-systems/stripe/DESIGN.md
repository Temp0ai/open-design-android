# Stripe Design System

## 1. Visual Theme & Atmosphere

Stripe's design embodies trust, innovation, and financial precision. The design system creates a sense of reliability and modern sophistication.

### Atmosphere
- Clean, trustworthy, premium
- Gradient-driven (signature purple-to-blue)
- Generous whitespace
- Subtle depth through shadows

## 2. Color Palette

### Primary Colors
- **Stripe Purple**: #635BFF
- **Stripe Cyan**: #00D4FF
- **Gradient**: linear-gradient(135deg, #635BFF, #00D4FF)

### Neutral Colors
- **Background**: #F6F9FC
- **Surface**: #FFFFFF
- **Border**: #E3E8EE
- **Text Primary**: #1A1F36
- **Text Secondary**: #697386
- **Text Tertiary**: #8792A2

### Accent Colors
- **Success**: #0EAD69
- **Warning**: #F5BE58
- **Error**: #DF1B41

## 3. Typography

### Font Family
- **Primary**: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif

### Scale
- **Display**: 48px / 56px / Bold
- **H1**: 36px / 44px / Bold
- **H2**: 28px / 36px / SemiBold
- **H3**: 22px / 28px / SemiBold
- **Body Large**: 18px / 28px / Regular
- **Body**: 16px / 24px / Regular
- **Small**: 14px / 20px / Regular

## 4. Spacing & Layout

### Spacing Scale (4px base)
- **0**: 0px
- **1**: 4px
- **2**: 8px
- **3**: 12px
- **4**: 16px
- **5**: 20px
- **6**: 24px
- **8**: 32px
- **10**: 40px
- **12**: 48px
- **16**: 64px

### Border Radius
- **Small**: 4px
- **Medium**: 6px
- **Large**: 8px
- **XLarge**: 12px

## 5. Components

### Buttons
- **Primary**: Purple gradient, white text, 4px radius
- **Secondary**: White background, purple border
- **Ghost**: Transparent, purple text

### Cards
- White background
- Subtle shadow: 0 2px 4px rgba(0,0,0,0.06)
- 8px radius
- 24px padding

## 6. Motion & Animation

### Easing
- **Default**: cubic-bezier(0.4, 0, 0.2, 1)

### Duration
- **Fast**: 150ms
- **Normal**: 250ms
- **Slow**: 400ms

## 7. Voice & Tone

- Professional but friendly
- Clear and direct
- Financially literate
- Innovation-focused

## 8. Anti-patterns

- Don't use flat colors without depth
- Don't skip the gradient
- Don't use harsh shadows
- Don't rush animations
